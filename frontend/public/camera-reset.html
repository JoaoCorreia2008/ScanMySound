🔄 Reset de Permissões

Esta página tenta limpar o estado de permissões guardado pelo Chrome para este site.

Se não funcionar automaticamente

No Chrome do telemóvel, abre este URL e remove o site:

chrome://settings/content/camera

Ou nas Definições do telemóvel:

  1. Aplicações → Chrome → Permissões → Câmara
  2. Remove qualquer entrada deste site
  3. Volta à app e tenta de novo
